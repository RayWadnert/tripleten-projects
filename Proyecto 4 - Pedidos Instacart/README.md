# Proyecto 4 — Pedidos Instacart (3M+ pedidos)

## 📌 Problema
Descubrir **segmentos de clientes** y **tendencias de recompra** para extraer insights de **retención** en un dataset grande.

## 📊 Dataset
- **Instacart** (dataset público)
- +**3,000,000** registros de pedidos/usuarios/productos
- Alto volumen → requiere **muestreo** o procesamiento eficiente

## 🛠️ Stack
- **Python:** pandas, matplotlib
- **Entorno:** Jupyter Notebook (`.ipynb`)

## 🔎 Metodología
1. **Muestreo** / carga por partes para manejar escala.  
2. **EDA** y creación de **features** (intervalos de recompra, tamaño de cesta).  
3. Gráficos de **tendencias de reorder** por segmento/categoría.  
4. (Opcional) Métricas de **retención** y ventanas temporales.

## 📈 Resultados / Insights
- Segmentos por **tendencia de recompra**.  
- Patrones en intervalos de reorden y categorías frecuentes.
